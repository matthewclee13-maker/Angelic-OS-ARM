# Angelic-OS 

My personal config of Angelic-OS

A text based Arch Linux distro made for the raspberry Pi 5. Angelic-OS has almost no GUI exept for a simple Waybar config and the Sway compositor is pre-installed for terminal emulators. For the times when having a GUI is just too much or when you want to look cooler than a Mac user.

comes pre-installed with the following applications:

hyprland,
neovim,
nano,
waybar,
kitty,
cool-retro-term,
btop,
yazi,
lazygit,
wiremix,
bluetui,
impala,
lynx browser,

keybinds:

super + K = keybinding guide 
super + return = open terminal window,
super + W = close window,
super + shift + b = lynx browser,
