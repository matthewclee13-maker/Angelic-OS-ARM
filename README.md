# Angelic-OS 

IMPORTANT! Angelic-OS is a work in progress and is not yet ready for daily use! Angelic-OS is only avalable for x86_64 but
is being ported over to ARM.
A text based Debian Linux distro made for the raspberry Pi 5. Angelic-OS has almost no GUI exept for a simple Waybar config and the Sway compositor is pre-installed for terminal emulators. For the times when having a GUI is just too much or when you want to look cooler than a Mac user.

comes pre-installed with the following applications:

plymouth,
hyprland
neovim,
nano,
waybar,
kitty,
btop,
yazi,
wiremix,
bluetui,
impala,

keybinds:

super + K = keybinding guide 
super + return = open terminal window,
super + W = close window,
super + shift + b = lynx browser,
